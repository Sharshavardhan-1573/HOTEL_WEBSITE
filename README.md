## HOTEL_WEBSITE
## 🏨 Cassaano Hotel Website

A modern, responsive luxury hotel website developed using **HTML5, CSS3, and JavaScript** as part of the **MANTRA 2026 Summer School Frontend Website Assignment**.

---

## 📖 Project Overview

Cassaano Hotel is a responsive frontend hotel website designed to provide visitors with an elegant browsing experience. The website showcases luxury rooms, hotel facilities, image galleries, and an online room booking form.

The project follows modern web development practices including responsive design, SEO optimization, and JavaScript interactivity.

## 📦 Installation

1. Download or clone the repository.

```
git clone https://github.com/your-username/cassaano-hotel.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

---

## 🌐 Deployment

### GitHub

1. Create a new repository.
2. Upload all project files.
3. Commit the project.

### Netlify

1. Login to Netlify.
2. Click **Add New Site**.
3. Import from GitHub or drag and drop the project folder.
4. Deploy the website.
